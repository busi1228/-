等保测评整改助手
一个单文件 HTML 工具，帮助网络安全等保测评人员对照 GB/T 22239-2019 标准进行差距分析和整改跟踪。支持 等保二级 和 等保三级 一键切换，覆盖全部 10 个安全层面。

功能特性
二级/三级双等级切换 — 顶部标签一键切换，二级 68 个控制项、三级 70 个控制项，数据按 GB/T 22239-2019 逐条编写
控制项符合性判断 — 每个控制项提供 ✅符合 / ⚠部分符合 / ❌不符合 / ⊘不适用 四种状态快捷按钮
展开选项联动 — 符合项和不符合项复选框独立勾选，勾选后自动填入结果记录区和整改建议区
内容全可编辑 — 复选框标签、textarea 文字均可手动修改，满足个性化定制需求
安全计算环境子模块 — 支持网络设备、安全设备、服务器、数据库、应用系统、终端、系统管理平台、关键数据 8 个子模块，每个子模块完整继承全部控制项
双层折叠 — 控制卡标题折叠 + 子项内容折叠，默认全部收起，浏览清晰
进度可视化 — 左侧边栏按章节显示完成百分比，三级子模块支持独立进度统计
导出 Excel 报告 — 一键导出结构化的测评结果和整改建议（基于 SheetJS）
报告预览与复制 — 右侧边栏实时预览报告，支持一键复制
数据持久化 — localStorage 自动保存，二级/三级数据完全隔离，互不影响
JSON 导入/导出 — 支持数据备份、迁移和团队共享
关键字搜索 — 支持按控制项标题和条目内容全文搜索
使用方式
直接在浏览器中打开 等保测评整改助手.html
默认进入等保二级，点击顶部 等保三级 切换等级
点击章节名切换到不同安全层面
点击控制项标题展开控制卡，点击子项标题展开详细选项
勾选符合/不符合项，或使用状态按钮快捷操作
所有内容可手动编辑修改
完成后导出 Excel 或复制预览报告
技术栈
纯静态 HTML + CSS + JavaScript，无框架依赖
SheetJS (xlsx) CDN 引入用于 Excel 导出
localStorage 实现数据持久化
完全离线可用（首次加载 SheetJS CDN 后）
