# 等保测评整改助手

一个纯前端单文件 HTML 工具，面向网络安全等保测评人员，对照 **GB/T 22239-2019** 标准进行差距分析和整改跟踪。支持等保二级与等保三级一键切换，覆盖全部 10 个安全层面。

## 功能特性

- **二级/三级双等级切换** —— 顶部标签一键切换，两套控制项数据完全独立，进度各自保存互不影响
- **四种符合性状态** —— 每个控制项支持 ✅ 符合 / ⚠ 部分符合 / ❌ 不符合 / ⊘ 不适用，一键勾选
- **复选框联动** —— 展开选项区分符合项和不符合项，勾选后自动填入结果记录区和整改建议区
- **全部内容可编辑** —— 复选框标签和 textarea 均支持手动修改，适应不同项目的个性化表述
- **安全计算环境子模块** —— 内置网络设备、安全设备、服务器、数据库、应用系统、终端、系统管理平台、关键数据 8 个子模块，完整继承全部控制项；支持用户自定义新建子模块
- **双层折叠** —— 控制卡标题 + 子项标签两级折叠，默认全部收起，浏览高效
- **章节进度条** —— 左侧边栏按章节显示完成百分比，三级支持子模块独立统计
- **Excel 导出** —— 基于 [SheetJS](https://sheetjs.com/) 一键导出结构化测评报告（控制项、结果记录、整改建议）
- **报告预览** —— 右侧边栏实时预览 Markdown 格式报告，支持一键复制
- **数据持久化** —— localStorage 自动保存，刷新不丢失；二级/三级数据按 key 隔离
- **JSON 导入/导出** —— 支持数据备份、迁移和团队共享
- **关键字搜索** —— 按控制项标题和条目内容全文检索

## 快速开始

1. 浏览器打开 `等保测评整改助手.html`
2. 默认显示等保二级，点击顶部 `等保三级` 切换等级
3. 左侧点击章节名称切换安全层面
4. 点击控制项标题展开控制卡，再次点击子项标签展开详细选项
5. 勾选符合/不符合项，或点击状态按钮快捷操作
6. 内容可手动编辑，实时自动保存
7. 完成后导出 Excel 或复制预览报告

## 控制项覆盖

所有控制项依据 **GB/T 22239-2019《信息安全技术 网络安全等级保护基本要求》**编写：

| 安全层面 | 二级子项数 | 三级子项数 | 三级主要增强 |
|----------|:--------:|:--------:|-------------|
| 安全物理环境 | 15 | 22 | 防盗报警、冗余供电、电磁屏蔽 |
| 安全通信网络 | 4 | 9 | 硬件冗余、传输加密 |
| 安全区域边界 | 11 | 19 | 应用层访问控制、内→外攻击防范 |
| 安全计算环境 | 22 | 34 | 双因素认证、数据保密性、异地备份 |
| 安全管理中心 | 4 | 12 | 集中管控、安全管理（三级独有） |
| 安全管理制度 | 6 | 7 | 制度体系细化 |
| 安全管理机构 | 9 | 14 | 专职安全管理员、审批留痕 |
| 安全管理人员 | 7 | 12 | 培训 ≥16 学时、脱密期管理 |
| 安全建设管理 | 23 | 35 | 第三方监理、源代码审计 |
| 安全运维管理 | 34 | 31 | 渗透测试、应急演练、变更管理 |

每个子项均包含：**符合项描述**（compliance）、**不符合项描述**（nonconform）、**整改建议**（suggestion），各 2–4 条。

## 技术栈

- 纯静态 HTML + CSS + JavaScript，零框架依赖
- [SheetJS (xlsx)](https://sheetjs.com/) CDN 引入，用于 Excel 导出
- localStorage 实现数据持久化
- 首次加载 SheetJS 后可完全离线使用
